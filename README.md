# Course & Records Holder (CRH)

Course & Records Holder (CRH) is a Java-based console application used to manage students, courses, enrollments, and grades.  
It allows users to import and export data, create backups, and generate reports using Java Streams.  
The system is designed using Object-Oriented Programming (OOP) principles, common design patterns, and modern Java features.
